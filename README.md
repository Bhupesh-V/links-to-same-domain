## Installation

 1. **Clone the repo**
  ```git clone https://github.com/Bhupesh-V/links-to-same-domain.git```
 2. **Change Directory**
  ```cd opengenus_internship_task/```
 3. **Install Dependencies**
 ```pip install -r requirements.txt```
 4. **Run Script**
  ```python task.py <WEBSITE_URL>```
